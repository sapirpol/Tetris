# Tetris Game in C++

## Game Controls

'y' - Up

'b' - Down (faster down)

'g' - Left

'h' - Right

'2' - Pause game

'a' - Change rotation (left/right)

'3' - speed

'4' - slow

'9' - exit

'1' - Restart Game


## Scoring

| Action           | Awarded points     |
|------------------|--------------------|
| Bomb             | num of blocks * 50 |
| clear using Joker| 50                 |
| 1 Row cleared    | 100                |
| 2 Row cleared    | 300                |
| 3 Row cleared    | 500                |
| 4 Row cleared    | 900                |
